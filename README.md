# Medical Consultation Platform

A modern and responsive medical consultation platform built with **React**, **React Router**, and **Bootstrap**. The application allows users to browse doctors, search by specialty, manage appointments, and experience a user-friendly authentication system.

## Features

* Home Page with medical platform overview
* Doctors Listing Page
* Search Doctors by name or specialty
* Doctor Details Page
* Appointment Booking System
* Appointment Confirmation Page
* My Appointments Management
* Favorite Doctors System
* User Registration and Login
* Authentication Validation
* Protected Routes
* Dynamic Navbar based on authentication status
* Logout Functionality
* Pagination for Doctors Listing (6 doctors per page)
* Dark Mode Support
* Medical Loading Screen
* Fully Responsive Design

## Technologies Used

* React
* React Router DOM
* Bootstrap 5
* Vite
* React Toastify
* LocalStorage

## Installation

```bash
npm install
npm run dev
```
