# Medical Consultation Platform

A modern medical consultation platform built with React, React Router, and Bootstrap.

## Features

- Home Page
- Doctors Listing
- Doctor Search
- Doctor Details Page
- Appointment Booking
- Appointment Confirmation
- Medical Loading Screen
- Responsive Design

## Technologies Used

- React
- React Router DOM
- Bootstrap
- Vite

## Installation

```bash
npm install
npm run dev
```
